# React Styling

## Description

A Holberton School project focused on styling a React application using **TailwindCSS v4**. Each task progressively converts the dashboard from plain CSS to Tailwind utility classes while keeping full test coverage.

## Project structure

| Directory | Description |
|-----------|-------------|
| `task_0/` | TailwindCSS v4 setup + Roboto font integration |
| `task_1/` | CourseList & CourseListRow Tailwind styles |
| `task_2/` | Notifications panel Tailwind styles |
| `task_3/` | Header, Login, BodySection & Footer Tailwind styles |
| `task_4/` | Responsive design with Tailwind breakpoints |
| `task_5/` | Bounce animation on the notifications title |

## Technologies

- React 18
- Vite 5
- TailwindCSS v4 (`@tailwindcss/vite`)
- `@fontsource/roboto` (weights 400, 500, 700)
- Jest + React Testing Library

## Requirements

- Node.js >= 18
- npm >= 9

## Getting started

```bash
cd task_0/dashboard
npm install
npm run dev
npm test
```
